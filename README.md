# Веб-сокеты и MQ-сервер

В приложении создана связка из броузера, веб-сокетов, MQ-сервера.
MQ-сервер рассылает сообщения напрямую подписавшимся броузерам.

Броузер с MQ взаимодействует через Stomp.
В качестве MQ-провайдера выступил ActiveMQ.

Также реализовано взаимодействие броузера с сервером не через Stomp, а через веб-сокеты, за это отвечает netty

В ActiveMQ версии 5.11.1 обнаружена ошибка, связанная с таймаутом веб-сокета, она уже исправлена в версии 5.12, 
но её пока нет в репозитории мавена.
